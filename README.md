## Background

Python programmers tend to not enjoy working with HTML and CSS - and who can blame them? Most of it is reused from project to project with just some basic changes. [Emmet](https://docs.emmet.io/cheat-sheet/) is a great web-developerâ€™s toolkit that takes the snippets idea to a whole new level. This project's goal is to create complete file templates and expand on Emmet snippets to eliminate as much HTML and CSS work as possible, maximizing Python programmers' use of time on actually programming in Python.

I will be updating this repository on regular basys.
